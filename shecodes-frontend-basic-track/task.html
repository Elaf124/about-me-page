Paragraph Display Difference

Each row uses real paragraph tags, but with different display values.

1) display: block

This is a real paragraph. Block starts on a new line and takes full width.

This is another real paragraph. Notice it appears on the next line.

2) display: inline

This is a real paragraph shown as inline, so it flows with text.

Second paragraph also stays in the same line when possible.

3) display: inline-block

This is a real paragraph as inline-block. It can keep box dimensions.

Second paragraph can sit beside the first one, like a box.
